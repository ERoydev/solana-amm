# Simple AMM Project

This repository contains a basic Automated Market Maker (AMM) implementation built for the Solana blockchain using the Anchor framework.

## Features
- Add liquidity to pools
- Swap tokens
- Built with Rust and Anchor

## Structure
- `anchor_project/` — Anchor smart contract and related scripts
- `frontend/` — Example React frontend for interacting with the AMM
- `backend/` — Rust backend utilities

## Getting Started
See the respective README files in each folder for setup and usage instructions.

---
This project is for educational purposes and demonstrates core AMM concepts on Solana.
